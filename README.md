SDG 12: AI-Driven Sustainable Waste Analytics 

              This project provides an end-to-end data pipeline to monitor, analyze, and predict global waste management patterns.
It is specifically designed to address **SDG Goal 12: Responsible Consumption and Production** using the 
[World Bank What a Waste Global Database](https://datacatalog.worldbank.org/search/dataset/0039597/what-a-waste-global-database).

 Key Features:
 
Automated Data Pipeline: Cleans and processes complex global waste datasets.
Machine Learning: Random Forest Classifier to predict regional "Sustainability Tiers."
Interactive Web App: Flask-based interface for real-time AI predictions.
BI Dashboard: Integrated Power BI report with geographic heatmaps and K-Means clustering visuals.

Tech Stack:

Language: Python 3.x
Libraries: Pandas, Scikit-learn, Matplotlib, Joblib
Web Framework: Flask
Visualization: Power BI Desktop

 Objective Coverage:
 
Monitor Patterns: Tracks waste per capita and total generation across 217 countries.
Management Insights: Deep-dive into recycling vs. dumping rates.
Promote Sustainability:Custom Sustainability Index ranking.
Predict Trends:ML-based classification of future sustainability levels.
