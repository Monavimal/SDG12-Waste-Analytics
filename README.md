SDG 12: AI-Driven Sustainable Waste Analytics

This project provides an end-to-end data pipeline to monitor, analyze, and predict global waste management patterns.
It is designed to support SDG Goal 12: Responsible Consumption and Production, using the
World Bank What a Waste Global Databasey.

 Key Features:
 
Automated Data Pipeline:
Cleans, transforms, and processes complex global waste datasets.

Machine Learning Model:
Uses a Random Forest Classifier to predict regional Sustainability Tiers.

Interactive Web App:
Flask-based interface for real-time predictions and insights.

BI Dashboard:
Power BI reports with geographic heatmaps and K-Means clustering visualizations.

 Tech Stack:
Language: Python 3.x
Libraries: Pandas, Scikit-learn, Matplotlib, Joblib
Web Framework: Flask
Visualization: Power BI Desktop

 Objective Coverage:
 
Monitor Patterns: Tracks waste per capita and total generation across 217 countries
Management Insights: Analyzes recycling vs. dumping rates
Promote Sustainability: Builds a custom Sustainability Index ranking
Predict Trends: ML-based classification of future sustainability levels

 Installation & Setup:
 
1. Clone the repository
git clone https://github.com/YOUR_USERNAME/SDG12-Waste-Analytics.git
cd SDG12-Waste-Analytics

3. Install dependencies
pip install -r requirements.txt

5. Run the Flask app
python app.py

 Impact:

This project demonstrates how AI + data analytics can help governments and organizations make informed decisions for sustainable waste management and environmental protection
